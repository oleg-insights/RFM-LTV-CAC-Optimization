# LTV/CAC оптимизация: снижение расходов и анализ лояльности

Цель проекта: 
Определить неэффективные расходы на привлечение клиентов путем оценки пожизненной ценности самых лояльных.

Ключевой вывод: 
LTV чемпионов в 9.34 раза выше, чем средний LTV остальных клиентов. Это обосновывает перераспределение бюджета от CAC к удержанию.

Гипотеза:
Выявленный дисбаланс в ценности групп позволяет безопасно забрать часть средств из привлечения новых клиентов и перенаправить ее на удержание лояльных. 

A/B-тест:
Для проверки эффективности перераспределения рекламного бюджета с приростом конверсии с 66% до 71% требуется выборка 2818 пользователей суммарно на две группы.
